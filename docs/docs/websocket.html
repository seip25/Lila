REAL-TIME ENGINE

WebSocket System

Lila Framework includes a Socket.io-style WebSocket engine built on top of Starlette with zero external dependencies. It provides room management, event broadcasting, fast orjson serialization, auto-reconnection, and client-side queuing.

⚡ High-Performance Backend

Uses WebSocketManager with binary orjson serialization, room channels, and automatic dead socket cleanup.

🔌 Smart Client (public/js/ws.js)

Zero-dependency Socket.io-style client (< 2 KB) with automatic exponential backoff reconnection and offline message queuing.

1. Backend Setup (lila/core/websocket.py)

Register WebSocket endpoints using @router.websocket("/ws") and manage events with WebSocketManager:

app/routes/web/chat.py 
from lila.core.routing import Router
from lila.core.websocket import WebSocketManager, WebSocket

router = Router()
ws_manager = WebSocketManager()

# Register custom event handler
@ws_manager.on("chat_message")
async def handle_chat(websocket: WebSocket, data: dict):
    # Broadcast to all connected clients
    await ws_manager.broadcast("chat_message", {
        "user": data.get("user", "Anonymous"),
        "text": data.get("text", "")
    })

# Register WebSocket endpoint
@router.websocket("/ws")
async def chat_socket(websocket: WebSocket):
    await ws_manager.handle_connection(websocket)

routes = router.get_routes()

2. Client-Side Usage (public/js/ws.js)

Include the built-in Lila WebSocket client script in your HTML template using {{ public('js/ws.js') }}:

resources/html/index.jinja 
<!DOCTYPE html>
<html>
<head>
  <title>Lila Real-Time Chat</title>
  <script src="{{ public('js/ws.js') }}"></script>
</head>
<body>
  <script>
    // Connect to WebSocket endpoint
    const ws = lilaWS('ws://' + location.host + '/ws');

    ws.on('connect', () => {
      console.log('Connected to Lila WebSocket!');
      ws.join('room-general'); // Join a room channel
    });

    ws.on('chat_message', (data) => {
      console.log('New message:', data.user, data.text);
    });

    // Send an event to the server
    function sendMessage(text) {
      ws.emit('chat_message', { user: 'Alice', text: text });
    }
  </script>
</body>
</html>

3. Rooms & Channels API Reference

  • ws_manager.join_room(websocket, "room_name") Subscribes a client connection to a specific channel/room.
  • await ws_manager.broadcast_to_room("room_name", "event", data) Sends a JSON event exclusively to clients in that room.
  • await ws_manager.emit_to(websocket, "event", data) Sends a JSON event to a single client socket.
