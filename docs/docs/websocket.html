REAL-TIME ENGINE

WebSocket System & Real-Time Chat

Lila Framework provides a zero-dependency, Socket.io-style WebSocket architecture. Powered by binary orjson serialization on the backend and an automatic reconnecting client (public/js/ws.js), building real-time applications, multi-room chats, and live dashboards is clean and simple.

⚡ Multi-Room Channels

Join, leave, and broadcast events to isolated rooms using broadcast_to_room().

🔄 Auto Reconnect & Queue

Client automatically reconnects with exponential backoff and queues offline messages until reconnected.

💓 Native Heartbeat

Built-in ping/pong heartbeat detects dropped client sockets quickly without consuming CPU.

1. Complete Real-Time Chat Backend (app/routes/web/chat.py)

Below is a complete, multi-room chat backend implementation using WebSocketManager:

app/routes/web/chat.py 
from lila.core.routing import Router
from lila.core.websocket import WebSocketManager, WebSocket
from lila.core.security import Security
import datetime

router = Router()
ws_manager = WebSocketManager()

@ws_manager.on("join_channel")
async def handle_join(websocket: WebSocket, data: dict):
    room = data.get("room", "general")
    username = Security.sanitize_data(data.get("username", "Guest"))
    
    ws_manager.join_room(websocket, room)
    
    # Notify room members that a new user joined
    await ws_manager.broadcast_to_room(room, "user_joined", {
        "user": username,
        "room": room,
        "time": datetime.datetime.now().strftime("%H:%M")
    })

@ws_manager.on("send_message")
async def handle_message(websocket: WebSocket, data: dict):
    room = data.get("room", "general")
    text = Security.sanitize_data(data.get("text", ""))
    username = Security.sanitize_data(data.get("username", "Guest"))
    
    if not text.strip():
        return

    # Broadcast message to room subscribers
    await ws_manager.broadcast_to_room(room, "new_message", {
        "user": username,
        "text": text,
        "room": room,
        "time": datetime.datetime.now().strftime("%H:%M")
    })

@router.websocket("/ws/chat")
async def chat_websocket_endpoint(websocket: WebSocket):
    """
    WebSocket endpoint handling lifecycle and event dispatching.
    """
    await ws_manager.handle_connection(websocket)

routes = router.get_routes()

    <!-- Connection Header -->
    <div class="flex items-center justify-between border-b border-slate-700 pb-4">
      <div class="flex items-center gap-2">
        <span id="status-badge" class="w-3 h-3 rounded-full bg-amber-500 animate-ping"></span>
        <h2 class="font-bold text-lg">Lila Chat Room (<span id="current-room" class="text-blue-400">#general</span>)</h2>
      </div>
      <select id="room-select" onchange="switchRoom(this.value)" class="bg-slate-900 border border-slate-700 rounded-xl px-3 py-1.5 text-sm">
        <option value="general"># general</option>
        <option value="tech"># tech</option>
        <option value="announcements"># announcements</option>
      </select>
    </div>

    <!-- Chat Messages Area -->
    <div id="messages-box" class="h-80 overflow-y-auto bg-slate-950 p-4 rounded-xl space-y-3 font-sans"></div>

    <!-- Input Area -->
    <form onsubmit="sendMessage(event)" class="flex gap-2">
      <input type="text" id="username" value="Alice" class="w-28 bg-slate-900 border border-slate-700 rounded-xl px-3 py-2 text-sm text-center" placeholder="Username" required />
      <input type="text" id="message-input" class="flex-1 bg-slate-900 border border-slate-700 rounded-xl px-4 py-2 text-sm" placeholder="Type a message..." required />
      <button type="submit" class="bg-blue-600 hover:bg-blue-500 text-white font-semibold px-5 py-2 rounded-xl text-sm transition-colors">Send</button>
    </form>
  </div>

  <script>
    let activeRoom = 'general';
    // Connect to WebSocket server
    const ws = lilaWS('ws://' + location.host + '/ws/chat');

    ws.on('connect', () => {
      document.getElementById('status-badge').className = 'w-3 h-3 rounded-full bg-emerald-500';
      // Join default channel
      ws.emit('join_channel', { room: activeRoom, username: getUsername() });
    });

    ws.on('disconnect', () => {
      document.getElementById('status-badge').className = 'w-3 h-3 rounded-full bg-rose-500';
    });

    // Listen for room broadcasts
    ws.on('new_message', (data) => {
      appendMessage(data.user, data.text, data.time, false);
    });

    ws.on('user_joined', (data) => {
      appendSystemMessage(`${data.user} joined ${data.room} at ${data.time}`);
    });

    function switchRoom(newRoom) {
      ws.emit('leave', activeRoom);
      activeRoom = newRoom;
      document.getElementById('current-room').innerText = '#' + newRoom;
      document.getElementById('messages-box').innerHTML = '';
      ws.emit('join_channel', { room: activeRoom, username: getUsername() });
    }

    function sendMessage(e) {
      e.preventDefault();
      const input = document.getElementById('message-input');
      const text = input.value;
      if (!text) return;

      ws.emit('send_message', { room: activeRoom, username: getUsername(), text: text });
      input.value = '';
    }

    function getUsername() {
      return document.getElementById('username').value || 'Guest';
    }

    function appendMessage(user, text, time) {
      const box = document.getElementById('messages-box');
      const div = document.createElement('div');
      div.className = 'flex flex-col bg-slate-900 border border-slate-800/80 p-3 rounded-xl max-w-md';
      div.innerHTML = `<div class="flex items-center justify-between text-xs text-slate-400 mb-1"><span class="font-bold text-blue-400">${user}</span><span>${time}</span></div><p class="text-sm">${text}</p>`;
      box.appendChild(div);
      box.scrollTop = box.scrollHeight;
    }

    function appendSystemMessage(text) {
      const box = document.getElementById('messages-box');
      const div = document.createElement('div');
      div.className = 'text-center text-xs text-slate-500 my-2 italic';
      div.innerText = text;
      box.appendChild(div);
      box.scrollTop = box.scrollHeight;
    }
  </script>
</body>

3. Summary API Reference

Backend (lila.core.websocket.WebSocketManager)

  • • @ws_manager.on("event"): Registers event callback.
  • • await ws_manager.broadcast("event", data): Sends event to all connected clients.
  • • await ws_manager.broadcast_to_room("room", "event", data): Sends event to room members.
  • • await ws_manager.emit_to(ws, "event", data): Sends event to a specific client socket.

Client (public/js/ws.js)

  • • const ws = lilaWS(url): Initializes WebSocket client instance.
  • • ws.on("event", callback): Listens for incoming event.
  • • ws.emit("event", data): Transmits event to server.
  • • ws.join("room") / ws.leave("room"): Joins or leaves a room channel.
